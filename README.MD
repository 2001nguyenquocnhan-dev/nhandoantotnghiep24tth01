# 🏪 Tiệm Tạp Hóa Bà Bảy - Project Website 2026

Chào mừng bạn đến với dự án website bán hàng tạp hóa trực tuyến. Đây là giao diện web được thiết kế tối ưu, thân thiện và mang đậm nét văn hóa Việt Nam.

## 🚀 Tính năng chính
* **Danh mục sản phẩm đa dạng:** Hơn 20 mặt hàng thiết yếu (Mì gói, Gia vị, Đồ uống, Hóa mỹ phẩm...).
* **Bộ lọc thông minh:** Lọc sản phẩm theo từng nhóm hàng nhanh chóng.
* **Giao diện Responsive:** Hiển thị đẹp mắt trên cả máy tính (Widescreen) và điện thoại.
* **Quản lý giỏ hàng:** Xử lý dữ liệu bằng JavaScript module (`cart.js`, `render.js`).

## 🛠 Công nghệ sử dụng
* **HTML5 & CSS3:** Cấu trúc và giao diện hiện đại.
* **JavaScript (ES6+):** Xử lý logic lọc hàng và giỏ hàng.
* **GitHub & Netlify:** Quản lý mã nguồn và triển khai trực tuyến.
